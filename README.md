#  IPL Performance Analysis (EDA) 🏏
IPL Performance Analysis using Python, Pandas, Matplotlib, and Seaborn with Exploratory Data Analysis (EDA), business questions, data visualization, and insights.
## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset using Python. The objective is to analyze player performance, team performance, match statistics, and discover meaningful business insights through data visualization.

---

## Business Problem

The IPL generates a large amount of match data every season. This project analyzes historical IPL data to identify player performances, team strategies, match outcomes, and other important insights that can support data-driven decision making.

---

## Objectives

This project answers the following business questions:

1. Which team won the most matches?
2. Which player scored the most runs?
3. Which bowler took the most wickets?
4. Which venue hosted the most matches?
5. Which teams won the most matches?
6. Which players hit the highest number of sixes?
7. Which bowlers conceded the fewest runs?
8. What is the distribution of runs scored?
9. Which players are the most consistent?
10. What are the key insights from the IPL dataset?
